# Expand

控制元素排列顺序，能够同时包含子级元素的排列顺序。

### 基础用法

::: demo
components/sort/demo1
:::

### 属性

| 参数     | 说明                        | 类型               | 默认值   |
| -------- | --------------------------- | ------------------ | -------- |
| sortList | 排列顺序                    | (number\|string)[] | []       |
| keyName  | 自定义节点排序属性的key名称 | string             | sort-key |
