## 属性

| 参数           | 说明                 | 类型    | 默认值 |
| -------------- | -------------------- | ------- | ------ |
| open           | 是否展开             | boolean | true   |
| targetRange    | 收起时的范围大小(px) | number  | 0      |
| transitionTime | 动画过渡时间(ms)     | number  | 200    |
