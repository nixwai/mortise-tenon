# 按钮

生成用于 `button` 的各种样式。

## 类型

提供了多种按钮类型，包括 `ghost`、`pale`、`soft`、`bright`、`deep`，默认为 `ghost`。

<demo vue="presets/button/types.vue"/>

## 大小

提供了 `xs`、`sm`、`md`、`lg` 四种大小，默认为 `md`。

<demo vue="presets/button/size.vue"/>
